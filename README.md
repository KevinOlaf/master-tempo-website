# Master Tempo Pro - Privacy Policy

Official privacy policy website for Master Tempo Pro iOS app.

**Live Site:** https://privacy.master-tempo.com (or your configured domain)

## Contents

- `index.html` - Privacy Policy page

## Hosting

Hosted on GitHub Pages with custom domain configuration.

## Updates

To update the privacy policy:
1. Edit `index.html`
2. Commit changes: `git add . && git commit -m "Update privacy policy"`
3. Push to GitHub: `git push`
4. Changes will be live within 1-2 minutes
